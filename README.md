## O projekte

Stránka vznikla s jednoduchým cieľom — mať všetky materiály prehľadne na jednom mieste, dostupné kedykoľvek a odkiaľkoľvek, pre mňa aj pre žiakov. Žiadne hľadanie súborov, žiadne sťahovanie — stačí otvoriť link.

Cvičenia sú interaktívne, fungujú priamo v prehliadači a sú prispôsobené konkrétnym témam a úrovniam. Materiály sú priebežne dopĺňané.
